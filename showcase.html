Project Overview

The Wearable Data Insight Generator transforms raw physiological data from wearable devices into personalized, actionable insights for athletes, health enthusiasts, and wellness professionals. By combining advanced feature engineering with domain-specific LLM fine-tuning, the system delivers contextually relevant recommendations that help users optimize their training, recovery, and overall health.

The latest version introduces Local LLM Support, allowing you to run cost-effective open-source models directly on your hardware without incurring API costs.

Local LLM Support

Run AI Models Locally

The latest update introduces support for running open-source language models directly on your hardware, eliminating the need for expensive API calls.

  • Cost-Effective: No more API costs for generating insights
  • Privacy-Focused: Your health data never leaves your device
  • Flexible Models: Choose from various Hugging Face models
  • Optimized Performance: 4-bit and 8-bit quantization for efficiency
  • Automatic Caching: Models are cached for improved performance

Getting Started

Installation

Follow these steps to set up the Wearable Data Insight Generator:


# Clone the repository
git clone https://github.com/teekag/wearable-llm-insight-generator.git
cd wearable-llm-insight-generator

# Create and activate a virtual environment
python -m venv wearable_venv
source wearable_venv/bin/activate  # On Windows: wearable_venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Set up environment variables
cp .env.example .env
# Edit .env with your configuration

# Run the application
python app.py

# Open your browser at http://localhost:8000
Requirements
  • Python 3.8+
  • 4GB+ RAM for local LLM
  • Internet connection for initial model download
  • Supabase account (optional)
